# 🚀 Jenkins CI/CD Pipeline

This project demonstrates a basic Continuous Integration (CI) pipeline using Jenkins and GitHub. Jenkins automatically pulls source code from a GitHub repository and executes build steps.

## 🛠️ Technologies Used

* Jenkins
* Git
* GitHub
* Python
* Windows

## ✨ Features

* Automated Build Execution
* GitHub Integration
* Continuous Integration Workflow
* Jenkins Freestyle Project
* Build Monitoring and Logs

## 📂 Project Structure

```text
jenkins-cicd-demo
│
├── app.py
├── requirements.txt
└── README.md
```

## 🔄 Workflow

1. Push code to GitHub Repository
2. Jenkins pulls the latest code
3. Build process starts automatically
4. Console Output is generated
5. Build status is displayed
